# 사전 평가 안내

<small>본 연구는 KAIST 생명윤리위원회에서 승인을 받은 연구입니다 (승인번호: KAISTIRB-2025-214).</small>

사전 평가는 총 X문항으로, Y분 정도 소요됩니다.
아래 "새 창에서 설문조사 열기"를 선택하여 별도의 창에서 응답하시거나, 이 페이지를 떠나지 않고 응답하실 수도 있습니다. 

모든 문항에 빠짐없이 답해주세요. 답안이 누락될 경우 다음 페이지로 넘어가지 않습니다.


⚠️ <span style={{color: 'red'}}>**주의: 설문 종료 시 답안 제출이 완료되었는지 꼭 확인해주세요.**</span>
<Steps>
    #### 설문을 완료하면, 우측 하단의 **화살표 버튼**을 눌러주세요.

    ![설문조사 제출 주의사항 1](https://www.dropbox.com/scl/fi/7jgvbeeoheaesczomwj0h/agreement_1.png?rlkey=zejvf6qlnbtegap8eux52d56x&raw=1)

    #### 응답 완료 화면이 나오는 것을 확인해주세요. 응답 완료 화면이 나오지 않았다면, 설문이 완료되지 않은 것입니다.
